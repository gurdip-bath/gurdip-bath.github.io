// What It Is

A production-grade Employee Lifecycle Hub built for a UK real estate company. Manages the full employee journey — onboarding, internal transfers, and offboarding — across HR Admin, HR Team Leader, HR Manager, and HR Business Partner personas.

Dataverse Power Automate Model-Driven App Canvas App Power BI Managed Solution

// Architecture Decisions

  • Dataverse over SharePoint — relational integrity, lookup enforcement, and RBAC at the data layer
  • Model-Driven App as primary HR interface — Dataverse-native, zero custom form code
  • Canvas App for line managers — filtered to direct reports only via User().Email
  • Import mode in Power BI — consistent performance over DirectQuery for scheduled reporting
  • Managed solutions for ALM — DEV changes never touch PROD directly

// Automated Flows

  • Generate Onboarding Tasks
  • Generate Offboarding Tasks
  • Stage-Change Notification
  • Overdue Case Reminder (Scheduled)

Every flow includes idempotency guards, per-iteration Try/Catch scoping, exponential backoff retry, and PII-compliant Secure Inputs/Outputs on email actions.

// Security Model

  • Four custom security roles scoped by Business Unit (HR and Finance child BUs)
  • Two Column Security Profiles restricting salary and sensitive HR fields
  • Secure Inputs and Outputs enabled on all email actions containing PII — UK GDPR compliant
  • Environment variables for all recipient addresses — no hardcoded values across environments
